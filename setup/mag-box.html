Troubleshooting

Portal stuck loading? Check Ethernet connection, confirm MAC is active, and power-cycle router plus box. If using Wi-Fi, prefer 5 GHz band.

Missing audio/desync? Switch Audio Output Mode to Auto (PCM) in settings. Update to the latest MAG firmware via Software Update.

Adult category visible? Use Parental Control menu to set a PIN and hide sensitive groups for family-safe setups.
