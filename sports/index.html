üßæ Event Schema Checklist

Use JSON-LD Event markup on every match page to secure rich results.

  • Start date/time: ISO8601 format with correct timezone (e.g., 2025-02-14T20:00:00+00:00).
  • Location: Use VirtualLocation pointing back to the landing page.
  • Offers: update to reflect free trial or subscription tiers if relevant.
  • Performer/competitor: include both teams or fighters for richer SERP context.
  • Update schedule: regenerate schema when kick-off times shift, especially for Champions League or UFC cards.
